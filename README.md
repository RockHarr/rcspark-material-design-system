# rcspark-material-design-system
Design system ligero basado en Bootstrap 5 e inspirado en Material 3. Pensado para aprender, experimentar y crear interfaces más limpias y accesibles.

> ⚠️ Nota
>
> Este proyecto se creó con fines de aprendizaje en el contexto del curso.
> Está pensado para que cualquiera pueda inspirarse, copiar partes,
> modificarlas y adaptarlas a sus propias tareas y proyectos.  
> La idea no es que todos entreguemos lo mismo, sino que tengamos una base
> más bonita y usable sobre la cual crear cosas propias.
